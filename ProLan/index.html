    表达式与语句

    表达式（Expression） 语句（Statement）

    >>> 2*6     # 经算术运算得到，可以求值
12      
>>> 7//4    # 除法取整运算
1
>>> 1 != 2  # Bool表达式
False

    以上三条均是表达式，其特点为可以求值，有算术和逻辑运算（基本表达式可以直接得到值而不需运算）

    >>> a = (100 + 50) // 2

>>> a
75

    一个赋值语句（Assignment Statement）由一个变量名和一个表达式组成。它会在**当前帧**下把变量名与该表达式的值绑定。
